# AI SMB Inquiry Assistant

This project is a prototype for an AI-powered assistant designed to handle inquiries for small and medium-sized businesses (SMBs).

## Features

- **Chat Interface**: A simple and intuitive web interface for users to interact with the AI.
- **OpenAI Integration**: Leverages powerful language models from OpenAI to understand and respond to user queries.
- **Configurable Models**: Easily switch between different OpenAI models (like GPT-4, GPT-4o-mini, etc.) via environment variables.

## Getting Started

### Prerequisites

- Python 3.x
- Node.js and npm

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Peter-Jiaqi-Han/ai-smb-inquiry-assistant.git
    cd ai-smb-inquiry-assistant
    ```

2.  **Setup Backend:**
    ```bash
    # Navigate to the backend directory
    cd backend

    # Install Python dependencies
    pip install -r requirements.txt
    ```

3.  **Setup Frontend:**
    ```bash
    # Navigate to the frontend directory
    cd ../frontend

    # Install Node.js dependencies
    npm install
    ```

### Running the Application

1.  **Start the backend server.**
2.  **Start the frontend development server.**
